TV Remote Control 技术支持

本应用目前仅支持可在本地网络开启 IP Control 或远程设备控制的兼容电视，并不支持所有电视。这里提供发现、配对、连接和遥控操作的常见问题处理方式。

使用前提

同一 Wi-Fi 网络

iPhone 需要和电视连接到同一个本地网络。访客网络、隔离网络或 VPN 可能导致发现失败。

电视端设置

请在电视设置中开启 IP Control、Remote Device Control，并按电视提示完成配对或认证设置。

本地网络权限

首次扫描时，请允许 iOS 的本地网络访问权限。关闭权限后，应用无法自动发现电视。

故障排查

无法发现电视

  1. 确认手机和电视在同一 Wi-Fi 网络。
  2. 确认 iOS 设置中已允许本地网络权限。
  3. 确认电视已开机，并且 IP Control 已启用。
  4. 如果自动发现仍不可用，请使用手动输入 IP。

配对或连接失败

  1. 检查电视屏幕上显示的配对码是否输入正确。
  2. 确认电视端允许新的远程控制设备连接。
  3. 如果使用手动 IP，请确认 IP 地址、端口和认证方式正确。
  4. 重启电视或重新扫描后再次连接。

遥控按键无响应

  1. 确认应用顶部显示的电视仍处于已连接状态。
  2. 确认电视未处于断网、休眠或输入源切换状态。
  3. 尝试返回设备管理重新连接。

反馈时请附带

  • 电视型号和系统版本。
  • iPhone 型号和 iOS 版本。
  • 连接方式：自动发现或手动 IP。
  • 问题截图、错误文案和复现步骤。
